Política de Privacidade

Controle Financeiro — Última atualização: 19 de maio de 2026

1. Introdução

O aplicativo Controle Financeiro respeita a sua privacidade. Esta política descreve como coletamos, usamos, armazenamos e protegemos suas informações.

2. Dados Coletados

2.1 Dados fornecidos pelo usuário

  • Informações da conta: nome, e-mail e senha (armazenada de forma criptografada) fornecidos no cadastro
  • Dados financeiros: valores, descrições, datas e categorias de salários, despesas, receitas, dívidas, cartão de crédito e metas de poupança

2.2 Dados coletados automaticamente

  • Dados de uso anônimos via Umami Analytics — sem cookies, sem identificação pessoal

3. Finalidade do Tratamento

  • Autenticar o usuário via e-mail e senha
  • Permitir o gerenciamento de finanças pessoais
  • Gerar resumos e relatórios financeiros
  • Melhorar a experiência do usuário

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros. Exceções:

  • Banco de dados MySQL (servidores seguros e criptografados)
  • Armazenamento local em arquivo JSON (servidor)
  • Umami Analytics (dados anônimos de uso)
  • Obrigação legal

5. Armazenamento e Segurança

  • Dados armazenados com criptografia em trânsito (TLS/SSL)
  • Senhas armazenadas com hash criptográfico (scrypt + salt)
  • Sessões autenticadas via JWT com token seguro
  • Acesso restrito ao usuário autenticado

6. Retenção e Exclusão

  • Dados mantidos enquanto a conta estiver ativa
  • Exclusão completa pode ser solicitada pelo app ou via e-mail em até 30 dias

7. Seus Direitos (LGPD)

  • Acessar seus dados pessoais
  • Solicitar correção de dados incompletos
  • Solicitar exclusão dos dados
  • Revogar o consentimento
  • Solicitar portabilidade
